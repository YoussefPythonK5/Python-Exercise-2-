# Age Classifier Script

### Description
This program solves the problem of automated age categorization to handle user data efficiently. It takes the user's age as an input and automatically determines whether they are an adult, a teenager, or a child based on predefined logical conditions.

### How to Run It
You can run this script on any Python compiler or IDE (like VS Code, PyCharm, or mobile compilers like Pydroid 3). Just execute the file, enter your age when prompted in the terminal, and press Enter.

### Tools Used
* **Programming Language:** Python
* **IDE/Compiler:** Pydroid 3 (used for development)
* **Core Concepts:** Input/Output functions (`input`, `print`) and Conditional Logic (`if`, `elif`, `else`).
